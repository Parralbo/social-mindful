# social-mindful
Mindful Across All Platforms .Control your social media usage across Facebook, Instagram, TikTok, YouTube, Reddit, and 6 more platforms. Pause, reflect, and choose how you spend your time with intention.

Multi-Platform Support: The extension now works seamlessly on 11 major social media platforms:
Facebook, Instagram, TikTok, Twitter/X, YouTube, Reddit, LinkedIn, Snapchat, Pinterest, WhatsApp Web, and Discord
Enhanced Features:
Platform-Specific Tracking: Each platform's usage is tracked separately, showing you exactly where your time goes
Unified Dashboard: The popup now displays usage statistics for all platforms with an intuitive tabbed interface
Smart Session Management: Sessions and blocks are managed independently per platform
Cross-Platform Insights: View which platforms consume the most of your time
Updated Website: The showcase website now highlights multi-platform support with a visual grid of all supported platforms and updated statistics sections.
